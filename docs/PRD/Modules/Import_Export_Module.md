# NoteWizard 模块文档：导入与导出 (Import & Export)

## 1. 功能概述
该模块负责笔记数据的跨平台迁移与第三方格式兼容，支持 NoteWizard 专属格式 (`.nwp`) 以及标准的 Markdown 格式。

## 2. NoteWizard 包 (.nwp)
- **定义**: 一种基于 ZIP 压缩的归档格式，包含完整的数据库结构（`objects/`, `nodes.jsonl`, `meta.json` 及 `images/`）。
- **导出逻辑**:
  - 扫描工作区 `Database` 目录。
  - 对于加密工作区，导出时会请求用户验证“恢复密钥”，以便将内容打包。
- **导入逻辑**:
  - 解压包内容到临时目录。
  - 增量合并 `nodes.jsonl`：如果存在 ID 冲突，则生成新的 ID 以防覆盖。
  - 物理移动 `objects/` 和 `images/` 文件到当前活跃工作区。

## 3. Markdown 导出与导入
- **导出**:
  - 将逻辑层级的笔记导出为真实的文件夹结构。
  - 自动处理正文中的本地图片引用，将其拷贝至对应层级的 `assets` 或 `images` 目录。
- **导入**:
  - 支持递归扫描文件夹。
  - 自动解析 Markdown 标题作为笔记名称。
  - 维护原始的目录层级关系。

## 4. 2.0 数据兼容性承诺
- **向前兼容**: v2.0 将继续沿用 v1.x 的 `Database` 目录结构（`nodes.jsonl` + `objects/`）。
- **无缝迁移**: Vue 版本的重构仅涉及渲染层逻辑，底层的虚拟文件系统 (VFS) 保持高度一致，确保用户升级后无需执行任何导出/导入即可直接访问历史笔记。
