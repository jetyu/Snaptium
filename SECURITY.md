# Security Policy

## Supported Versions

The following versions of **Notewizard** are currently supported with security updates.

| Version        | Supported |
| -------------- | --------- |
| Latest  | ✅ |
| Older  | ❌ |

> Only the latest released version receives security fixes.  
> Please upgrade to the latest version to ensure you receive security updates.

---

## Reporting a Vulnerability

If you discover a security vulnerability, please **DO NOT OPEN A PUBLIC ISSUE**.

Instead, report it responsibly using the following methods:

- Open a **GitHub Security Advisory** for this repository  

### What to expect

- You can expect an initial response within **72 hrs**
- We will investigate and validate the reported issue
- If the vulnerability is confirmed, we will work on a fix and release an update
- Proper credit will be given to the reporter when appropriate

Thank you for helping keep Notewizard and its users safe.
