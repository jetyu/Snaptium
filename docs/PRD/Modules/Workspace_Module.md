# NoteWizard 模块文档：工作区与文件管理 (Workspace)

## 1. 虚拟文件系统 (VFS)
- **元数据存储**: 基于 `db.json`（或 `metadata.json`）维护笔记的层级结构、名称、创建时间、修改时间及加密状态。
- **物理分离**: 笔记内容存储在 `objects/` 目录下，以 UUID 或哈希值命名，逻辑名称与物理路径解耦，便于移动和重命名而不损坏链接。

## 2. 树状导航 (Tree View)
- **多级嵌套**: 支持无限层级的文件夹（笔记本）管理。
- **操作交互**:
  - **内联重命名**: 点击菜单触发，直接在树节点上通过 input 修改名称。
  - **拖放移动 (Drag & Drop)**: 支持节点之间的层级移动及排序。
  - **上下文菜单**: 右键支持新建笔记、新建笔记本、重命名、删除（移至回收站）、锁定、显示物理文件位置、查看属性。

## 3. 搜索系统
- **实时过滤**: 支持通过搜索框对工作区内的所有笔记标题进行实时模糊匹配。
- **全文搜索**: (v1.2 新增) 支持搜索笔记内容，检索结果高亮显示。

## 4. 状态恢复
- **节点记忆**: 启动时自动选中最后一次编辑的笔记（若存在）。
- **折叠状态**: 记忆文件夹的展开/折叠状态。

## 5. 文件属性
- **字数统计**: 自动计算并显示总字符数（去除空白）。
- **行数统计**: 显示文件总行数。
- **物理追溯**: 显示文件的物理路径及修改时间记录。
