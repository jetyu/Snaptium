「Apple 风格 AI Workspace Dashboard」

整体设计语言：

macOS Dashboard 风格
Notion + Linear + Arc Browser 的融合感
大留白
超轻边框
半透明柔光
模块化卡片
极简信息层级

它不是后台管理系统。

更像：

一个“正在工作的 AI 知识空间”。

一、整体布局结构

整体是：

┌─────────────────────────────┐
│ 顶部导航栏                   │
├────────┬────────────────────┤
│ 左侧栏  │ 主工作区            │
│ Sidebar│                    │
└────────┴────────────────────┘

比例：

Sidebar：16%
Main：84%

风格：

非常薄
半透明
macOS Toolbar 感

结构：

[ Logo ]      [ Search ]      [ Theme ] [ Notification ] [ Avatar ]
设计特点
1. 搜索栏居中

非常 Apple。

像：

Spotlight
Arc
Raycast

特点：

超浅背景
无边框
高度低
圆角大
2. 右侧状态图标

采用：

极简线性 Icon
小尺寸
大间距

避免：

彩色图标
高对比
