# NoteWizard 模块文档：UI/UX 框架 (UI Framework)

## 1. 布局架构 (Layout)
- **两分栏/三分栏**: 
  - 左侧：树状图 (Tree) 与 搜索 (Search)。
  - 中间：Markdown 编辑器 (Editor)。
  - 右侧：实时预览 (Preview) 与 大纲 (Outline/TOC)。
- **可调节面板**: 支持通过拖拽边缘调整各面板宽度，状态持久化到配置中。

## 2. 大纲导航 (Outline / TOC)
- **动态提取**: 实时解析 Markdown 中的 `#` 标题层级。
- **交互逻辑**:
  - **单点击**: 编辑器自动滚动到对应行并聚焦。
  - **双点击**: 折叠/展开该层级的大纲子项。
  - **自动高亮**: 编辑器滚动时，大纲自动高亮当前阅读位置对应的标题。

## 3. 回收站机制 (Trash)
- **逻辑删除**: 节点标记 `trashed: true`，物理文件移动至 `Database/trash`。
- **管理界面**:
  - 提供文件列表，支持“恢复”或“永久删除”。
  - “清空回收站”将物理删除 `trash/` 下的所有文件并清理 `nodes.jsonl` 中的标记。

## 4. 模态框与对话框 (Modals)
- **统一风格**: 包含首选项、恢复密钥输入、加密提示等。
- **ESC 绑定**: 全局监听 ESC 键关闭活跃模态框。
