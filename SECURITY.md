# Security Policy / 安全策略

## Supported Versions / 支持的版本

The following versions of **Notewizard** are currently supported with security updates.  
以下版本目前仍提供安全更新：

| Version | Supported |
| ------ | --------- |
| Latest | ✅ |
| Older  | ❌ |

> Only the latest version receives security fixes.  
> 仅最新版本提供安全更新，请及时升级。

---

## Reporting a Vulnerability / 报告安全漏洞

If you discover a security vulnerability, please **do not open a public issue**.  
如果你发现了安全漏洞，请 **不要直接创建公开 Issue**。

Instead, report it responsibly using one of the following methods:  
请通过以下方式私下报告：

- Open a **GitHub Security Advisory** (Recommended)  
  通过 GitHub 安全公告提交（推荐）

You can expect an initial response within **72 hours**.  
我们将在 **72 小时内**给予初步回复。

Thank you for helping keep Notewizard secure.  
感谢你帮助提升 Notewizard 的安全性。
